🚀 一键启动客户端

Linux / macOS
Windows (PS3+)
Win 7/8 (PS2)
📦 curl 一键启动（无需安装）
curl -sL http://HOST/install.sh | sh -s -- ws://HOST
wget 版（无 curl 时）
wget -qO- http://HOST/install.sh | sh -s -- ws://HOST
⚙️ 自定义 ID 和密码
curl -sL http://HOST/install.sh | sh -s -- ws://HOST -i my-device -p secret
📦 PowerShell 3.0+（Win10/11）
$u='http://HOST/install.ps1';iwr $u -OutFile $env:TEMP\rdev.ps1;& $env:TEMP\rdev.ps1 ws://HOST
⚙️ 自定义 ID 和密码
$u='http://HOST/install.ps1';iwr $u -OutFile $env:TEMP\rdev.ps1;& $env:TEMP\rdev.ps1 ws://HOST -Id my-pc -Password secret
📦 PowerShell 2.0（Win7/8）
$wc=New-Object Net.WebClient;$wc.DownloadFile('http://HOST/install.ps1',"$env:TEMP\rdev.ps1");& "$env:TEMP\rdev.ps1" ws://HOST
⚙️ 自定义 ID 和密码
$wc=New-Object Net.WebClient;$wc.DownloadFile('http://HOST/install.ps1',"$env:TEMP\rdev.ps1");& "$env:TEMP\rdev.ps1" ws://HOST -Id my-pc -Password secret

⚡ SSH 连接方式

ssh <deviceID>@<server> -p SSHPORT scp file <deviceID>@<server>:/tmp/ -P SSHPORT sftp -P SSHPORT <deviceID>@<server>

🔀 端口转发

# -L 访问设备上的服务 ssh -L 8080:localhost:80 <deviceID>@<server> -p SSHPORT # -R 暴露本地服务到服务器 ssh -R 3000:localhost:3000 <deviceID>@<server> -p SSHPORT

🔑 免密登录

# 将公钥加入服务端 ~/.rdev/authorized_keys cat ~/.ssh/id_ed25519.pub >> ~/.rdev/authorized_keys ssh <deviceID>@<server> -p SSHPORT # 无需密码!
