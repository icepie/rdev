📦 一键安装并启动
curl -sL http://___SERVER___/install.sh | sh -s -- ws://___SERVER___
📦 wget 版 (无 curl 时)
wget -qO- http://___SERVER___/install.sh | sh -s -- ws://___SERVER___
⚙️ 自定义参数
curl -sL http://___SERVER___/install.sh | sh -s -- ws://___SERVER___ -i my-device -p secret
📦 PowerShell 3.0+ (Win10/11)
$u='http://___SERVER___/install.ps1';iwr $u -OutFile $env:TEMP\rdev.ps1;& $env:TEMP\rdev.ps1 -Server ws://___SERVER___
⚙️ 自定义参数
$u='http://___SERVER___/install.ps1';iwr $u -OutFile $env:TEMP\rdev.ps1;& $env:TEMP\rdev.ps1 -Server ws://___SERVER___ -Id my-pc -Password secret
📦 PowerShell 2.0 (Win7/8)
$wc=New-Object Net.WebClient;$wc.DownloadFile('http://___SERVER___/install.ps1',"$env:TEMP\rdev.ps1");& "$env:TEMP\rdev.ps1" -Server ws://___SERVER___

⚡ SSH 连接方式

ssh <deviceID>@<server> -p 2222 scp file <deviceID>@<server>:/tmp/ -P 2222 sftp -P 2222 <deviceID>@<server>

🔀 端口转发

# -L 本地转发: 访问设备上的服务 ssh -L 8080:localhost:80 <deviceID>@<server> -p 2222 # -R 远程转发: 暴露本地服务到服务器 ssh -R 3000:localhost:3000 <deviceID>@<server> -p 2222

🔑 免密登录

# 将公钥加入服务端 ~/.rdev/authorized_keys cat ~/.ssh/id_ed25519.pub >> ~/.rdev/authorized_keys ssh <deviceID>@<server> -p 2222 # 无需密码!
